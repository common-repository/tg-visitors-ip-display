Visitors IP display

== Description ==
This Plugin displays the IP of the visitors in the website. Even if the visitor is using proxy pass, This IP will fetch and display the real IP of the visitor.

== Installation ==
1. Upload \"tgvisitorsipdisplay.php\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Place short code [tg_ip_display] in pages/posts/widget text area and you are done.
